Alumni Portal

Complete Documentation & Setup Guide

Welcome!

This comprehensive alumni portal helps you connect alumni, manage events, share opportunities, and build a thriving community.

Installation

1
Navigate to the project folder 
cd alumni-portal
2
Install dependencies 
npm install
3
Start the server 
npm start
4
Open in browser 
http://localhost:3000

Features

Alumni Directory

Search and connect with alumni by name, batch, or department

Events

Create and manage reunions, seminars, and networking events

Job Board

Post and browse career opportunities from the community

News & Updates

Share achievements and important announcements

Profile Management

Maintain professional profiles with career information

Secure Authentication

Password hashing and session-based security

Technology Stack

  • Backend: Node.js
  • Framework: Express.js
  • Template: EJS
  • Frontend: Bootstrap 5
  • Icons: Bootstrap Icons
  • Auth: bcryptjs
  • Storage: JSON Files
  • Session: Express-session
  • Parser: body-parser

Getting Started

First Time Setup:
  1. Register a new account by clicking "Register" in the navigation
  2. Fill in your personal and educational details
  3. Login with your credentials to access the dashboard
  4. Complete your profile with professional information
  5. Start exploring and connecting with other alumni!

Customization

Change Server Port:

Edit server.js and modify:

const PORT = 3000; // Change to your preferred port
Modify Departments:

Edit views/register.ejs to update the department dropdown options.

Upgrade to Database:

Replace JSON file storage with MongoDB or PostgreSQL for production use.

Security Features

  • Password hashing using bcryptjs for secure storage
  • Session-based authentication with secure cookies
  • Protected routes requiring authentication
  • Input validation on forms
